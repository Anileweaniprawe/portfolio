Parkinson's Detection from Voice Biomarkers

EngineeringThesis · ongoing

A reproducible machine learning workflow for classifying Parkinson's disease from repeated voice recordings.

Approach: Built a subject-level evaluation pipeline to prevent recordings from the same person appearing in both training and test data. I compare SVM, Random Forest and XGBoost, test preprocessing and class-balancing strategies, and use SHAP to examine which voice features influence model predictions.

The next stage will include cross-lingual external validation on an independent dataset recorded in another language to assess how well the identified voice biomarkers generalise.

Key focus: Leakage-safe validation, realistic performance estimation, model interpretability and cross-lingual generalisation.

Thyroid Disease Prediction

2026
Machine Learning Course · Dublin City University

An end-to-end classification project for highly imbalanced tabular data.

Approach: Cleaned and explored the UCI Sick Thyroid dataset, compared five classifiers and applied SMOTE only within cross-validation folds. We tuned the decision threshold using F-beta to prioritise recall, engineered an additional clinical ratio feature and used SHAP to interpret the final model.
